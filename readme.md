# Codeigniter 3 - Natural Sort

Codeigniter Natural Sort with CRUD Data

![Screenshot](/screenshot/1.png)

### Description
Show Data from Database with 2 Comparation
- Original Sorting by Name (Ascending)
- Natural Sorting
